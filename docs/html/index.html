Parse¶

class csgo.parser.demoparser.DemoParser(demofile='', outpath=None, log=False, demo_id=None, parse_rate=128, parse_frames=True, trade_time=5, dmg_rolled=False, buy_style='hltv')¶

This class can parse a CSGO demofile to various outputs, such as JSON or CSV. Accessible via csgo.parser import DemoParser

demofile¶

A string denoting the path to the demo file, which ends in .dem

Type:string
log¶

A boolean denoting if a log will be written. If true, log is written to “csgo_parser.log”

Type:boolean
demo_id¶

A unique demo name/game id. Default is inferred from demofile name

Type:string
parse_rate¶

One of 128, 64, 32, 16, 8, 4, 2, or 1. The lower the value, the more frames are collected. Indicates spacing between parsed demo frames in ticks. Default is 128.

Type:int
parse_frames¶

Flag if you want to parse frames (trajectory data) or not

Type:bool
trade_time¶

Length of the window for a trade (in seconds). Default is 5.

Type:int
dmg_rolled¶

Boolean if you want damages rolled up (since multiple damages for a player can happen in 1 tick from the same weapon.)

Type:bool
buy_style¶

Buy style string, one of “hltv” or “csgo”

Type:string
Raises:ValueError – Raises a ValueError if the Golang version is lower than 1.14
parse(return_type='json')¶

Wrapper for _parse_demo() and _read_json(). Provided for user convenience.

Parameters:return_type (string) – Either “json” or “df”
Returns:A dictionary of output
